Minha Solução Desafio 02 - 7DayCode - Alura

Explicação da Solução

O objetivo deste desafio é capturar informações do usuário e exibir uma mensagem personalizada com base nas respostas fornecidas. Além disso, um questionamento extra avalia o interesse do usuário na linguagem escolhida.

Passos da Implementação:

  • Solicita o nome do usuário.
  • Solicita a idade do usuário.
  • Solicita a linguagem de programação que o usuário está estudando.
  • Exibe uma mensagem personalizada com as respostas capturadas.
  • Pergunta se o usuário gosta da linguagem e exibe uma resposta condicional.

Exemplo de Implementação:

                
                    let nome = prompt("Qual o seu nome?");
                    let idade = prompt("Quantos anos você tem?");
                    let linguagem = prompt("Qual linguagem de programação você está estudando?");

                    alert(`Olá ${nome}, você tem ${idade} anos e já está aprendendo ${linguagem}!`);

                    let resposta = prompt(`Você gosta de estudar ${linguagem}? Responda com 1 para SIM ou 2 para NÃO.`);
                    
                    if (resposta == 1) {
                        alert("Muito bom! Continue estudando e você terá muito sucesso.");
                    } else if (resposta == 2) {
                        alert("Ahh que pena... Já tentou aprender outras linguagens?");
                    } else {
                        alert("Resposta inválida. Por favor, responda com 1 ou 2.");
                    }
